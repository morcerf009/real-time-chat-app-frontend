# Smart Bot | Premium Real-Time Chat Interface

A beautiful, responsive, and persistent chat application built with vanilla HTML, CSS, and JavaScript. Features a simulated AI bot ("Smart Bot"), glassmorphism UI design, and local storage integration.

[![Live Demo](https://img.shields.io/badge/demo-online-green.svg)](https://deft-toffee-5241aa.netlify.app/)

## âœ¨ Features

- **Premium UI Design**: Modern glassmorphism aesthetic with smooth animations and responsive layout.
- **Real-Time Simulation**: Experience instant messaging with a simulated "Smart Bot" that replies naturally.
- **Smart Persistence**: Chat history is automatically saved to your browser's `localStorage`, so your conversation is never lost on refresh.
- **Interactive Elements**:
  - **File Attachment**: Simulate file uploads with visual feedback and bot acknowledgement.
  - **Clear History**: Safely clear your chat with a custom confirmation modal.
- **Privacy First**: All data is stored locally on your device.
